<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with TailorHub Skills in minutes
  </p>

  <Steps>
    <Step title="Install a skill">
      Run the CLI with npx to install any skill from the TailorHub repository:

      ```bash
      npx @tailorhub/skills@latest add https://github.com/TailorHub-Mad/ai-skills/tailor-code-review
      ```

      By default, the skill installs to both Claude Code and Codex. Use `--target` to specify a single environment.
    </Step>

    <Step title="Restart your AI coding assistant">
      After installing a skill, restart Claude Code or Codex to load the new capabilities.

      <Note>
        Skills are installed to `~/.claude/skills/` and `~/.codex/skills/` by default.
      </Note>
    </Step>

    <Step title="Use the skill">
      Invoke the skill by name in your AI assistant. For example, with the code review skill installed:

      ```
      Review my current branch with tailor-code-review
      ```

      The AI assistant will execute the skill workflow and provide results.
    </Step>
  </Steps>
</div>
